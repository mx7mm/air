## Decisions Made

### 1. Level 1 Custom OS
We're building on the Linux kernel, not writing our own. This gives us:
- Hardware support out of the box
- Years of security hardening
- Focus on what matters: the user experience

### 2. Technology Stack
| Component | Choice | Reason |
|-----------|--------|--------|
| Kernel | Linux | Hardware support, stability |
| libc | musl | Small, clean, modern |
| Graphics | Wayland | Modern, secure, simple |
| Compositor | Custom (wlroots) | Full control over UX |
| Build System | Buildroot | Proven, flexible |

### 3. Documentation First
Every term, decision, and build attempt gets documented. No tribal knowledge.
